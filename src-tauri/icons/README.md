# Icons

Place app icons here. Required files (generated by Tauri's `tauri icon` command):

- `32x32.png`
- `128x128.png`
- `128x128@2x.png`
- `icon.icns` (macOS)
- `icon.ico` (Windows)

Quickly generate from a single 1024x1024 source:

```sh
npx @tauri-apps/cli icon path/to/source-1024.png
```

Until you provide a source icon, builds may emit warnings but `tauri dev` still runs.
